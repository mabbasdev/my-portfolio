Project Overview

The goal was to build a clean, user-friendly site to reflect elegance of the personal brand. This project involved UI/UX planning, design execution in Figma, and frontend development using HTML5, CSS and JavaScript, also involving smooth animations for the softness.

The Story

The inspiration from different portfolio websites let me design this sleek digital space that felt like a premium brochure. After a few competitor analysis, I mapped out the structure and wireframe that would define the website's clean and intentional aesthetics.

Our Approach

Starting out with a wireframe-first approach and moved into visual design and typography refinement using Figma. Smooth transitions were added using prototypes to elevate the user experience without overcomplicating it. Turned my aesthetic figma visuals into website design using HTML5, CSS JavaScript, implementing my best practices possible.
